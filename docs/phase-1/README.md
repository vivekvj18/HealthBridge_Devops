# Phase 1 Baseline

This folder captures the Phase 1 groundwork for the health-bridge microservices split.

## Deliverables in this phase

- external API baseline for the current monolith
- package-to-service boundary mapping
- internal API conventions for future service-to-service routes
- Actuator health/info endpoint setup
- environment-specific Spring profile files
- initial `libs/health-bridge-common` scaffold
